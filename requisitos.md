RF01 - Cadastro de Usuário

O sistema deve permitir que o usuário realize cadastro informando login e senha.

RF02 - Autenticação

O sistema deve permitir login e logout do usuário.

RF03 - Cadastro de Receitas

O usuário deve poder registrar receitas com descrição, valor, data e categoria.

RF04 - Cadastro de Despesas

O usuário deve poder registrar despesas com descrição, valor, data e categoria.

RF05 - Edição de Registros

O sistema deve permitir editar receitas e despesas cadastradas.

RF06 - Exclusão de Registros

O sistema deve permitir excluir receitas e despesas.

RF07 - Categorias

O sistema deve permitir criar, editar e excluir categorias.

RF08 - Visualização de Saldo

O sistema deve exibir o saldo atual do usuário.

RF09 - Relatórios

O sistema deve gerar relatórios financeiros por período (mensal/semanal).

RF10 - Filtro por Data

O sistema deve permitir filtrar registros por período.
