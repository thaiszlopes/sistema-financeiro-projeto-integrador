# 💰 Sistema de Gestão Financeira Pessoal

## 📌 Descrição

Este projeto tem como objetivo desenvolver um sistema simples de gestão financeira pessoal, permitindo o controle de receitas, despesas e geração de relatórios.

## 🎯 Objetivo

Auxiliar usuários a organizar suas finanças e tomar decisões mais conscientes.

## ⚙️ Funcionalidades

* Cadastro e login de usuários
* Registro de receitas e despesas
* Categorias personalizadas
* Visualização de saldo
* Relatórios financeiros

## 🧱 Tecnologias Utilizadas

* Frontend: HTML, CSS, JavaScript
* Backend: Python (Flask/Django)
* Banco de Dados: SQLite / Firebase
